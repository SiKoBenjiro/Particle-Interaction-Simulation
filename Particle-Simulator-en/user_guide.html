User Guide - Particle Simulator

Getting Started

To create a new particle simulation:

  1. Enter the particle parameters (position, charge, mass, etc.)
  2. Click "Add Particle" to add it to the simulation
  3. Set simulation constants (G, k, time)
  4. Click "Run Simulation" to start

Visualization Options

The simulator offers several visualization modes:

  • Trajectory Lines - Shows the path of particles
  • Density Heatmap - Shows where particles spend most time
  • X-Axis Histogram - Distribution along X axis
  • Y-Axis Histogram - Distribution along Y axis

Saving and Loading

You can save your particle configurations and load them later using the Save/Load buttons.
